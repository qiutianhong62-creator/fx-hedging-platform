# 企业外汇套保与风险分析平台

面向出口企业的交互式外汇套保产品与风险分析项目。当前 MVP 接入远期结汇，支持自定义美元敞口、远期汇率、套保比例和到期即期汇率，并实时生成人民币收入情景曲线。

## 当前功能

- 远期结汇收益计算
- 0%—100% 自定义套保比例
- 套保与不套保收益曲线对比
- 到期汇率情景分析
- 人民币收入保护与机会成本解释
- Python 计算引擎及单元测试
- 为期权、掉期和货币市场套保预留产品接口

## 核心公式

```text
最终人民币收入
= 美元敞口 × 套保比例 × 远期汇率
+ 美元敞口 × (1 - 套保比例) × 到期即期汇率
```
