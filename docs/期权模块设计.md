# 外汇期权模块设计

> 文档版本：V0.1  
> 当前范围：欧式期权到期收益与企业现金流  
> 标的与报价：美元，人民币/美元（CNY per USD）

## 1. 模块目标

期权模块首先解决“到期时这项期权和企业原有敞口合在一起会产生多少人民币现金流”的问题。

第一版不预测汇率，也不直接计算银行应当报出的期权价格。复杂定价、波动率和Greeks将在到期收益逻辑稳定后加入。

期权由两个相互独立的维度组成：

1. **期权类型**：看涨期权（Call）或看跌期权（Put）。
2. **持仓方向**：买入期权（Long/Buy）或卖出期权（Short/Sell）。

因此基础期权头寸共有四种：买入看涨、卖出看涨、买入看跌、卖出看跌。

## 2. 第一版统一假设

- 标的是美元，汇率 `S` 表示1美元对应多少人民币。
- 名义金额 `N` 使用美元表示。
- 执行价 `K` 使用人民币/美元表示。
- 期权费率 `p` 使用人民币/美元表示，期权费总额为 `N × p`。
- 第一版使用欧式到期期权，只在到期日判断收益。
- 第一版将期权费直接计入最终现金流，暂不考虑期权费从支付日至到期日的资金时间价值。
- 第一版使用现金等价的到期收益，不处理真实交割、保证金和交易对手信用风险。

后续如果使用其他货币对，必须先明确标的货币和报价方向，不能直接套用美元/人民币的文字解释。

## 3. 看涨与看跌

### 3.1 美元看涨期权

美元看涨期权给予买方按照执行价买入美元的权利。

单位内在价值：

```text
max(S - K, 0)
```

当美元到期汇率高于执行价时，看涨期权具有内在价值。

### 3.2 美元看跌期权

美元看跌期权给予买方按照执行价卖出美元的权利。

单位内在价值：

```text
max(K - S, 0)
```

当美元到期汇率低于执行价时，看跌期权具有内在价值。

## 4. 买入与卖出

| 头寸 | 买方或卖方的核心含义 | 期权费 |
|---|---|---|
| 买入看涨 | 有权按执行价买入美元 | 支付期权费 |
| 卖出看涨 | 如果对方行权，有义务按执行价卖出美元 | 收取期权费 |
| 买入看跌 | 有权按执行价卖出美元 | 支付期权费 |
| 卖出看跌 | 如果对方行权，有义务按执行价买入美元 | 收取期权费 |

买方拥有权利而没有必须行权的义务；卖方收取期权费，同时承担被行权后的履约义务。

## 5. 四类头寸的到期净收益

以下公式只表示期权头寸本身，不包括企业原有的美元应收或应付现金流。

### 5.1 买入看涨

```text
N × max(S - K, 0) - N × p
```

盈亏平衡汇率：

```text
K + p
```

### 5.2 卖出看涨

```text
-N × max(S - K, 0) + N × p
```

盈亏平衡汇率同样为 `K + p`。

### 5.3 买入看跌

```text
N × max(K - S, 0) - N × p
```

盈亏平衡汇率：

```text
K - p
```

### 5.4 卖出看跌

```text
-N × max(K - S, 0) + N × p
```

盈亏平衡汇率同样为 `K - p`。

## 6. 与企业敞口结合

期权不能只看独立收益，还要与企业原有的应收或应付敞口合并。

### 6.1 美元应收：出口企业

不套保的人民币收入为：

```text
N × S
```

最典型的保护方式是买入美元看跌期权：

```text
最终人民币收入
= N × S + N × max(K - S, 0) - N × p
= N × max(S, K) - N × p
```

它为美元贬值设置收入下限，同时在扣除期权费后保留美元升值带来的收入机会。

出口企业也可能卖出美元看涨期权，收取期权费但让渡部分美元升值收益：

```text
最终人民币收入
= N × S - N × max(S - K, 0) + N × p
= N × min(S, K) + N × p
```

单独卖出看涨期权会产生重要义务和风险。更常见的企业风险管理用途是将其与买入看跌期权组合成区间或领式策略。

### 6.2 美元应付：进口企业

不套保的人民币成本为：

```text
N × S
```

最典型的保护方式是买入美元看涨期权：

```text
最终人民币成本
= N × S - N × max(S - K, 0) + N × p
= N × min(S, K) + N × p
```

它为美元升值设置购汇成本上限，同时在扣除期权费后保留美元贬值带来的成本下降机会。

进口企业也可能卖出美元看跌期权，收取期权费但让渡部分美元贬值带来的成本下降机会：

```text
最终人民币成本
= N × S + N × max(K - S, 0) - N × p
= N × max(S, K) - N × p
```

更常见的企业用途是将卖出看跌与买入看涨组合成进口企业的区间策略。

## 7. 企业场景对应关系

| 企业敞口 | 主要担心 | 基础保护期权 | 主要效果 |
|---|---|---|---|
| 美元应收 | 美元贬值、结汇收入下降 | 买入美元看跌 | 设置人民币收入下限 |
| 美元应付 | 美元升值、购汇成本上升 | 买入美元看涨 | 设置人民币成本上限 |
| 美元应收 | 希望收取期权费，愿意让渡部分上涨收益 | 卖出美元看涨 | 限制收入上方空间 |
| 美元应付 | 希望收取期权费，愿意让渡部分成本下降收益 | 卖出美元看跌 | 限制成本下方空间 |

其他组合并非绝对禁止，但系统必须说明其与企业敞口的匹配关系，并对可能形成的额外方向性风险发出提示。

## 8. 风险边界

| 期权头寸 | 最大收益与损失特征 |
|---|---|
| 买入看涨 | 期权头寸最大损失为期权费；汇率上升时收益增加 |
| 卖出看涨 | 最大收益为期权费；汇率持续上升时可能产生很大的履约损失 |
| 买入看跌 | 期权头寸最大损失为期权费；汇率下降时获得保护 |
| 卖出看跌 | 最大收益为期权费；汇率大幅下降时可能产生显著履约损失 |

平台展示的“最大损失”必须区分期权头寸本身与“期权＋企业敞口”组合，不能把两者混为一谈。

## 9. 期权模块输入字段

- 企业敞口方向：美元应收或美元应付
- 期权类型：看涨或看跌
- 持仓方向：买入或卖出
- 美元名义金额
- 覆盖比例
- 执行价
- 期权费率或期权费总额
- 期权费币种
- 到期日
- 到期即期汇率或情景范围
- 结算方式：第一版使用现金等价结算

## 10. 输出字段

- 期权内在价值
- 毛期权收益
- 期权费现金流
- 净期权现金流
- 不使用期权时的企业基准收入或成本
- 使用期权后的最终人民币收入或成本
- 相对基准的保护收益或机会成本
- 盈亏平衡汇率
- 到期收益曲线
- 风险说明

## 11. 复合策略示例

### 11.1 出口企业领式策略

```text
买入较低执行价的美元看跌期权
＋ 卖出较高执行价的美元看涨期权
```

目标是在降低或抵消期权费的同时，建立收入下限和上方收益边界。

### 11.2 进口企业领式策略

```text
买入较高执行价的美元看涨期权
＋ 卖出较低执行价的美元看跌期权
```

目标是在降低或抵消期权费的同时，建立成本上限和下方成本边界。

### 11.3 远期与期权组合

```text
部分敞口使用远期锁定
＋ 部分敞口使用保护性期权
＋ 剩余部分不套保
```

未来组合引擎应将每个组成项的现金流相加，在同一张图中与纯远期、纯期权和不套保策略比较。

## 12. 分阶段开发

### 第一阶段：到期现金流

- 四类基础头寸
- 欧式到期收益
- 期权费
- 应收和应付敞口
- 情景曲线
- 风险提示

### 第二阶段：组合策略

- 多个期权组成项
- 领式和区间策略
- 远期与期权组合
- 覆盖比例和重复覆盖检查

### 第三阶段：期权定价

- Black-Scholes模型
- 国内外无风险利率
- 隐含波动率
- Delta、Gamma、Vega、Theta和Rho
- 波动率曲面

### 第四阶段：高级分析

- 蒙特卡洛模拟
- 美式期权二叉树模型
- 压力测试
- 市场报价与模型价格比较
